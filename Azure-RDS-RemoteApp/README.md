# Azure Remote Desktop Services (RDS) RemoteApp Deployment

## Overview

This project demonstrates the deployment of Microsoft Remote Desktop Services (RDS) on Microsoft Azure using Windows Server 2022.

The objective was to publish WordPad as a RemoteApp through RD Web Access, allowing users to access only the application instead of a complete remote desktop session.

## Technologies Used

- Microsoft Azure
- Windows Server 2022
- Remote Desktop Services (RDS)
- RD Session Host
- RD Web Access
- RD Connection Broker
- Azure Virtual Network
- Network Security Group (NSG)

## Features

- Azure Virtual Machine Deployment
- Static Public IP
- DNS Configuration
- RemoteApp Publishing
- RD Web Access
- Security Configuration
- Deployment Validation
